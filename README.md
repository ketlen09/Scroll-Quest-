
# 🎮 Scroll Quest

## 📖 Sobre o Projeto

Scroll Quest é um jogo criado no Scratch onde o jogador controla um aventureiro que precisa explorar um labirinto em busca de 8 pergaminhos escondidos.
Cada pergaminho contém uma pergunta sobre o Brasil. Ao responder corretamente, o jogador ganha pontos e continua sua jornada até encontrar todos os pergaminhos.

## 🎯 Objetivo

* Encontrar os 8 pergaminhos espalhados pelo labirinto.
* Responder corretamente às perguntas sobre o Brasil.
* Conseguir o maior número de acertos possível.
* 
## 🕹️ Como Jogar

*Use as setas do teclado para mover o aventureiro.
*Explore o labirinto procurando os pergaminhos.
*Ao tocar em um pergaminho, responda à pergunta apresentada.
*Cada resposta correta aumenta sua pontuação.
*Encontre todos os pergaminhos para concluir a aventura.
## 📚 Perguntas do Jogo

* Qual é a capital do Brasil?
* Quantos estados o Brasil possui?
* Qual é a moeda do Brasil?
* Qual é a maior floresta do Brasil?
* Em qual continente fica o Brasil?
* Qual é a língua oficial do Brasil?
* Qual é o rio mais famoso da Amazônia?
* Qual é a cor predominante da bandeira do Brasil?
